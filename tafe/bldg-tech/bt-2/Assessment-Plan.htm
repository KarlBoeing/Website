ASSESSMENT STRUCTURE

This module has four (4) Assessment elements

  •  Assessment 1
        Synopsis Submission (Scope of work) 
  •  Assessment 2
        Draft Assignment Submission
  •  Assessment 3
        Final Assignment Submission
  •  Assessment 4
        Oral Presentation      and written Summary Pages

    • A pass in each of the four (4) elements of Assessment is required to pass the Subject. If a pass is not achieved in any of the four assessment elements at first attempt, an 'H' grade is to be applied. The student must submit or sit tests for second attempt to clear the 'H' grade. If the student does not PASS the second attempt within the Re-sit Week (Week 20) an 'R' grade is to be applied.

    NOTE: .
    Candidates will need to demonstrate competence a number of times under various conditions to satisfy the Learning outcomes/Assessment Criteria. Under CBT and AQTF provisions candidates do not receive a percentage mark for the module completed. Candidates demonstrating competence will receive a grade of CO (competent).

    Candidates failing to demonstrate competence at the end of an individual assessment task will receive the grade of H (Hold), and will be required to resubmit the assessment. If a pass is not achieved in any of any Assessment at first attempt, an 'H' grade is to be applied. Candidates must resubmit the Assignment within seven (7) days to clear the 'H' grade.

    If the student does not PASS the second attempt within the stipulated resubmission time frame an 'R' grade is to be applied. Candidates failing to demonstrate competence at the end of the module will receive the grade of R (re-enrol), and will be required to re-enrol in the module again and complete ALL assessment tasks again to the required competence level.
