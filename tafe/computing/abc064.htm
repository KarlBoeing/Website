MODULE TITLE                  Building Computing Applications 1

 

Nominal duration                   35 - 40 hours.

 

Module codes

            DETAFE(SA)             CDAH

            National                      ABC064

 

Discipline code                       0502320

Module purpose                     To provide and develop student skills in a range of computing software applications that will compliment experience gained in other Modules.

 

Relationship to competency standards

                                                No Competency Standards are currently available.

 

Prerequisites                          None.

 

Summary of content               Word processing                              

                                                Spread sheets                                   

                                                Estimating and Costing                       ? Currently using Excel

                                                Scheduling/Project Management        ? Currently using Excel

                                                Computer Aided Drafting                   Not covered

                                                Reference software eg `BC Aider'      using LRC Web page

 

Delivery                                  Students must have use of a Personal Computer with current software

 

Assessment method               Practical exercises and tests

 

Conditions of assessment      It is assumed that most of the training related to this module will be off the job.

                                                Learners will be required to demonstrate safe work practices.

                                                Learners will be provided with a list of possible sources of information and references.

                                                During assessment students shall have access to permitted references.

 

Learning outcomes                On completion of this module, students will be conversant with the industrial application and use of the computing software programs listed below;

 

Students should use the Menu bar and the Start button only to perform the task outlined in the document.

(All tool bars (especially Standard and Formatting) should be turned off and not be shown on screen. (This method works well and helps you to get a better understanding of how to navigate through the program. It also achieves better outcome than using toolbars.)
