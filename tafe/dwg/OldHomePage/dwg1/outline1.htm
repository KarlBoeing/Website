COURSE OUTLINE

 

Module Title:             BUILDERS’ WORKING DRAWING 1

 

Nominal Duration:     60 Hours

 

Module Code:            ABC061                                 W.A. SIN:   27 742

 

Module Purpose:       To provide the participants with basic skills required to read and interpret plans and specifications and undertake basic architectural drafting.

 

Co-Requisite:             Construction 1

 

Text Required:           Architectural Drawing Standards and Conventions

                                    Basic Training Manual 14.10, PLAN READING.

 

Reference Texts:       Appropriate Australian Standards; Glossary of Building and Planning Terms; AS 1100 Architectural Drawing and Supplement 1.

 

Course Content

Synopsis:                    Sizes of sheets, paper types, Use of drawing equipment, pens and pencils, Lettering and Lines; Symbols; Scaling; Drawing and dimensioning of details; Plan Reading; A part set of working drawings for single storey building.

 

Assessment

Details:                       This module has five (5) assessment elements; one (1) written plan reading Assignment PLUS four drawing Assignments.

 

                                    Assessment percentage to be the accumulation of marks scored in each of the Assignments.

 

                                    Value of Assignments:

 

                                    Lettering and Scaling- Assignment 1      =          20%

                                    Symbols - Assignment 2                       =          20%

                                    Details - Assignment 3                          =          20%

                                    Building Drawing - Assignment 5           =          30%

                                    Plan Reading - Assignment 4                 =          10%

                                                                                    Total   =          100%

                                    A pass in each of the five (5) elements of Assessment is required to pass the Module.  A pass is defined as 50% of each elements’ percentage value.

.

                                    If a pass is not achieved in any of Assignments 1, 2, 3, 4, 5 at first attempt, an ‘H’ grade is to be applied.  The student must submit a second attempt at the same Assignment and PASS within the enrolment period or an ‘R’ grade is to be applied.

 

Assessment

Methods:                    Plan Reading (Assignment 4) to be from B.T.M. 14.10, plan reading exercises 1 to 5 inclusive.  An assessment comprised of selected questions from the exercises will be carried out.

 

                                    Drawing Assignments (Assignments 1, 2, 3, 5) to be completed in ink using drafting pens on tracing paper or film and submitted with a print copy to the lecturer by the nominated date given for each Assignment.

 

Learning Outcomes:  See National Module ABC 061 (S.I.N. 27742) documentation, and Programme for delivery.
