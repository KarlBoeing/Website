INSTRUCTIONS TO STUDENTS:

The aim of this module is to extend the students knowledge of basic measuring and levelling techniques to include instruments and calculations necessary in the set out and control of large residential developments, commercial and industrial construction.

a)   This module is designed to be carried out over one Semester at 1.5 hours each class.

b)   There are four assessments to be submitted. For the field surveying the class will be divided into groups to carry out three individual assignments

c)   The task is based on a TEAM effort and it is the responsibility of each student in the group to maintain this perspective (refer to key competencies).

TASK:

A range of surveying techniques used for commercial building applications will be explored. A theodolite will be used for the measurements of vertical and horizontal angles (bearings). Polar coordinates as well as chain lines and offsets with an optical square are utilised for set out buildings.

 

Exercise 1 - Supervised field work

With aid of a chain line and offsets determine the outlay of a rectangular building. Locate wooden pegs on the corner of the building

Surveying field notes need to be recorded for work done in the field. Make sure all measurements are correctly recorded in your field notes.

Each student must submit:

·        a clear description of the field exercise

·        the field records (chain line & offsets)

·        a plan scale 1:50, including house dimensions

Exercise 2 – Supervised field work

In this exercise we use a theodolite to
(a) peg out a rectangular building with given polar coordinates

(b) to locate features on a building site.

Use the theodolite determine horizontal bearing angles and distances. The theodolite should be set & up in an appropriate position to measure the horizontal angles and distances to trees.

Each student must submit:

   ·     a clear description of the field exercise

   ·     the records taken in the field (polar coordinates)

   ·     the calculation of all measurements between the surveyed trees

   ·    a plan scale 1:200, showing the position of the trees.

    

Assignment 2 - Field work without supervision

This assignment is similar to Exercise 1. You will use the offset method to set out a more complex building. The building set out offsets measured at right angles to the chain line and inclined offsets are used to locat the trees as schown in the instruction sheet.
This assignment must be submitted in Week 14 (29 October 2008)

Each student must submit:

·        a clear description of the field exercise (a precise outline of work you do)

·        the field records (original measurenents on chain line & offsets)

·        a plan scale 1:50, including all dimensions of the house

Assignment 3- Field work without supervision

This assignment is similar to Exercise 2. You will use the theodolite to:

(a) set out a building lot given polar coordinates,
(b) find the height RL's of the ridge of the Trafo Station (RL's at ground and ridge level)

This assignment must be submitted in Week 18 (26 November 2008)

Each student must submit:

·        a clear description of the field exercise (a precise outline of work you do)

·        the field records (pegging out of the building lot including height measurements)

·        a plan scale 1:50, including all side dimensions.

(area & perimeter of the lot, all corner angles & sketch for RL's at Trafo Station )
